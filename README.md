# 📲 WhatsApp Bulk Messaging System (Automated Notification Tool)

## 📌 Description
This project is a WhatsApp Bulk Messaging System that enables users to send messages to multiple contacts efficiently using API integration. It supports contact management, customizable messages, and automation features, helping businesses streamline communication while reducing manual effort.

---

## 🚀 Features
- Send messages to multiple contacts at once  
- Upload contacts (CSV / database)  
- Custom message templates  
- Fast and automated messaging  
- Simple and user-friendly interface  

---

## 🛠️ Tech Stack
- Frontend: HTML, CSS, JavaScript  
- Backend: PHP  
- Database: MySQL  
- API: WhatsApp API / Third-party API
